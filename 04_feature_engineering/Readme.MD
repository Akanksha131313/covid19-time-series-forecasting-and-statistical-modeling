**04 – Feature Engineering for Predictive Modeling**
----------------------------------------------------------------------------------------------------------------------------------------------------------------

**Objective -**

Construct predictive features capturing temporal dependency, volatility, growth momentum, and population-normalized severity to enhance forecasting model performance.

----------------------------------------------------------------------------------------------------------------------------------------------------------------

**Engineered Feature Categories -**

• Lag Features (1, 3, 7-day dependency)

• Rolling Mean (7 & 14-day smoothing)

• Rolling Volatility (Standard Deviation)

• Growth Acceleration

• Population-Normalized Metrics

• Calendar-Based Variables

-------------------------------------------------------------------------------------------------------------------------------------------------------------------

**Key Impact -**

• Captures autoregressive behavior

• Reduces noise sensitivity

• Detects surge momentum

• Improves cross-state comparability

• Enables model generalization

------------------------------------------------------------------------------------------------------------------------------------------------------------------
**Outcome -** Created a structured, model-ready feature matrix designed to improve forecasting stability and predictive accuracy.
