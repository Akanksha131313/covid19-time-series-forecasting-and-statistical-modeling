**India COVID-19 Statewise Time-Series Dataset (2021) -** 

---------------------------------------------------------------------------------------------------------------------------------------------------------------

**Overview-**

• This dataset provides a structured, state-level daily COVID-19 time-series view for India covering 365 days across 36 States/UTs.

• It is designed for:

a) Time-series forecasting

b) Regression modeling

c) Public health risk analysis

d) Resource allocation simulations

e) Clustering & pattern detection

f) Total Records: 13,140 rows

------------------------------------------------------------------------------------------------------------------------------------------------------------------

**Why This Dataset Exists -

-> Most publicly available COVID datasets are either:

• Cross-sectional (one snapshot)

• Poorly structured for ML workflows

• Missing daily granularity

-> This dataset solves that by providing:

• Daily state-level progression

• Cumulative and incremental metrics

• Clean numeric formatting

• ML-ready schema

----------------------------------------------------------------------------------------------------------------------------------------------------------------

**Dataset Structure -** 

Column	                                                     Description

a) Date	                                                     Reporting date

b) State_UT	                                                 Indian State / Union Territory

C) Population	                                               State population

d) New_Cases	                                               Daily confirmed cases

e) New_Deaths	                                               Daily reported deaths

f) New_Recoveries	                                           Daily recoveries

g) Total_Cases	                                             Cumulative confirmed cases

h) Total_Deaths	                                             Cumulative deaths

i) Total_Recoveries	                                         Cumulative recoveries

j) Active_Cases	                                             Active cases (derived)

----------------------------------------------------------------------------------------------------------------------------------------------------------------

**Analytical Use Cases -**

-> This dataset supports -

• Predicting next-day or next-week case growth

• Identifying high-risk states using mortality ratio trends

• Modeling healthcare burden via active case trajectories

• Comparing population-normalized infection spread
