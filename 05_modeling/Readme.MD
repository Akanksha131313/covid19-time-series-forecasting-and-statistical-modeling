**05 – Predictive Modeling**

-----------------------------------------------------------------------------------------------------------------------------------------------------------------

**Objective -**

Develop supervised learning models to predict daily COVID-19 new cases using lag-based and epidemiological indicators.

-------------------------------------------------------------------------------------------------------------------------------------------------------------------

**Modeling Approach -**

• Chronological train-test split

• Baseline Linear Regression

• Random Forest for non-linear capture

• Evaluation via MAE, RMSE, and R²

------------------------------------------------------------------------------------------------------------------------------------------------------------------

**Key Findings -**

• Random Forest improved predictive stability over linear baseline.

• Lag-1 and Lag-7 new cases were primary predictive drivers.

• Active cases significantly influenced short-term case growth.

-----------------------------------------------------------------------------------------------------------------------------------------------------------------

**Impact -**

• Built leakage-safe modeling pipeline

• Identified key case growth drivers

• Established benchmark vs non-linear improvement

• Created foundation for advanced forecasting
